strSort R Documentation

Sort characters in a string

Description

Alphabetically sorts characters in a string. Vectorized over x.

Usage

strSort(x)

Arguments

x

A string to sort.

Value

A sorted string.

Examples

strSort("cba")
strSort("zyxcCbB105.a")
strSort(c("cba", "zyx"))
strSort(c("cba", NA))
