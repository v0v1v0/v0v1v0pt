has_names R Documentation

Has or lacks names?

Description

has_names tests if an object has names
lacks_names tests if an object lacks names

Usage

  has_names(x)

Arguments

x

an R object

See Also

has_rownames

Examples

set.seed(1)
x <- y <- runif(10)
names(x) = letters[1:10]

has_names(x) # TRUE
has_names(y) # FALSE

lacks_names(x) # FALSE
lacks_names(y) # TRUE
