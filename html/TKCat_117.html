Description

Get instance timestamps of an MDB in chTKCat

Usage

get_chMDB_timestamps(x, name)

Arguments

x

a chTKCat object
name

the name of the database

Value

A tibble with the instance "timestamp" and a logical indicating if it's the "current" one or not.
