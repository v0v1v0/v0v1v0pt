dos2unix R Documentation

Convert Line Endings

Description

Convert line endings in a text file between Dos (CRLF) and Unix (LF) format.

Usage

dos2unix(file)

unix2dos(file)

Arguments

file

a filename.

Value

No return value, called for side effects.

See Also

line.endings examines line endings.

write.taf uses unix2dos to ensure that the resulting files have Dos line endings.

TAF-package gives an overview of the package.

Examples

## Not run: 
file <- "test.txt"
write("123", file)

dos2unix(file)
file.size(file)

unix2dos(file)
file.size(file)

file.remove(file)

## End(Not run)
