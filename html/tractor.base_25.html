fx R Documentation

Shorthand anonymous functions

Description

These functions provide a shorthand route to simple anonymous functions.

Usage

fx(expr)

fxy(expr)

fxyz(expr)

fi(expr)

Arguments

expr

A (single or compound) expression forming the body of the function.

Value

The function constructed.
