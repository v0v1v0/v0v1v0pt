movie_alternative_title R Documentation

Retrieve alternative titles for movies.

Description

Get the alternative titles for a specific movie id.

Usage

movie_alternative_title(api_key, id, country = NA, append_to_response = NA)

Arguments

api_key

Your TMDb Consumer Key.

id

The movie ID.

country

A ISO 3166-1 code.

append_to_response

Comma separated, any movie method.

Value

A list with the following fields:

id

The movie ID.
titles

The alternative titles of the movie.

Author(s)

Andrea Capozio

References

https://developers.themoviedb.org/3/getting-started
