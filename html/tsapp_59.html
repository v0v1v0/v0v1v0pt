periodotest R Documentation

periodotest computes the p-value of the test for a hidden periodicity

Description

periodotest computes the p-value of the test for a hidden periodicity

Usage

periodotest(y)

Arguments

y

vector, the time series

Value

pval the p-value of the test

Examples

  
data(PIGPRICE)
y <- PIGPRICE
out <- stl(y,s.window=6)  
e <- out$time.series[,3]
out <- periodotest(e)
