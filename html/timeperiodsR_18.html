last_workday R Documentation

Get last workday day in period.

Description

Method for get last workday in timeperiod.

Usage

last_workday(x)

Arguments

x

tpr object or date vector

Value

Vector of dates

Author(s)

Alexey Seleznev

Examples

pm_last_workday <- last_workday(pm)
