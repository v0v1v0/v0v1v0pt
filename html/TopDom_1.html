countsPerRegion R Documentation

Calculates Counts per Region in a TopDomData Object

Description

Calculates Counts per Region in a TopDomData Object

Usage

countsPerRegion(data, regions)

Arguments

data

A TopDomData object.
regions

TopDom regions (a data.frame), e.g. domains.

Value

A numeric vector of length nrow(regions).
