as.lagpol R Documentation

Lag polynomial

Description

as.lagpol converts a numeric vector c(1, -a_1, ..., -a_d) into a lag polynomial (1 - a_1 B - ... - a_p B^p).

Usage

as.lagpol(pol, p = 1)

Arguments

pol

a numeric vector.
p

integer power.

Value

An object of class lagpol.

Examples

as.lagpol(c(1, -0.8))
as.lagpol(c(1, 0, 0, 0, -0.8))
