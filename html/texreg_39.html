extract,forecast_ARIMA-method R Documentation

extract method for forecast_ARIMA objects

Description

extract method for forecast_ARIMA objects created by the Arima function in the forecast package.

Usage

## S4 method for signature 'forecast_ARIMA'
extract(
  model,
  include.pvalues = TRUE,
  include.aic = TRUE,
  include.aicc = TRUE,
  include.bic = TRUE,
  include.loglik = TRUE,
  include.nobs = TRUE,
  ...
)

Arguments

model

A statistical model object.
include.pvalues

Report p-values?
include.aic

Report Akaike's Information Criterion (AIC) in the GOF block?
include.aicc

Report AICC in the GOF block?
include.bic

Report the Bayesian Information Criterion (BIC) in the GOF block?
include.loglik

Report the log likelihood in the GOF block?
include.nobs

Report the number of observations in the GOF block?
...

Custom parameters, which are handed over to subroutines. Currently not in use.
