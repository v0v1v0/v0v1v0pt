caes R Documentation

Construct aesthetic mappings

Description

Low-level version of ggplot2::aes.

Usage

caes(x, y, ...)

Arguments

x, y, ...

List of name-value pairs in the form aesthetic = variable.

Value

a list of quosure.

Examples

caes(x = month, y = value)
caes(x = month, y = value, fill = city)
