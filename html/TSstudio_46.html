zoo_to_ts R Documentation

Converting 'zoo' object to 'ts' object

Description

Converting 'zoo' object to 'ts' object

Usage

zoo_to_ts(zoo.obj)

Arguments

zoo.obj

a univariate 'zoo' object

Examples


data("EURO_Brent", package = "TSstudio")
class(EURO_Brent)
ts_plot(EURO_Brent)
EURO_Brent_ts <- zoo_to_ts(EURO_Brent)
class(EURO_Brent_ts)
ts_plot(EURO_Brent_ts)
