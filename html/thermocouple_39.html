SelfHeatingError R Documentation

self-heating error

Description

SelfHeatingError calculates the self-heating error

Usage

 SelfHeatingError(I, R, Ek)

Arguments

I

intensity (A)
R

resistance (ohm)
Ek

self-heating coefficient(mW/C)

Value

E

self-heating error
