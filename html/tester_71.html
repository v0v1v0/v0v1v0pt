is_single R Documentation

Is single

Description

Tests if an object is single (i.e. of length 1)

Usage

  is_single(x)

Arguments

x

an R object

See Also

is_single_number, is_single_string, is_single_logical

Examples

is_single("hoskdflksfd")  # TRUE
is_single("1.0")  # TRUE
is_single(1:5)  # FALSE
is_single(matrix(runif(4), 2, 2))  # FALSE
