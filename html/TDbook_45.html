data-text_RMI_tree R Documentation

Tree to display with symbolic points indicating partitioned bootstrap values

Description

This tree is to display with symbolic points with different color or shape indicating the bootstrap values belong to which category.
Catagories are defined by cutting the bootstrap values into several interval to indicate whether the clade is high, moderate or low support.
Chap.13.2

Format

character

Examples

data(text_RMI_tree)
