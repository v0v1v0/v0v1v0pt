tm1_get_log R Documentation

TM1 Get Logs of an instance

Description

Gets server logs from a tm1 instance

Usage

tm1_get_log(tm1_connection, lognumber)

Arguments

tm1_connection

tm1 connection object returned by the function tm1_connection
lognumber

Number of how many lines of logs you want. Default is 5

Examples

## Not run: 
tm1_get_log(tm1_connection("localhost", "8881", "admin", "apple"), 10)

con_obj <- tm1_connection("localhost", "8881", "admin", "apple")
tm1_get_log(con_obj)

## End(Not run)
