is_cwl R Documentation

Is this a CWL object?

Description

Is this a CWL object?

Usage

is_cwl(x)

Arguments

x

any object

Value

Logical. TRUE if it is a CWL object, FALSE if not.

Examples

system.file("cwl/sbg/workflow/gatk4-wgs.json", package = "tidycwl") %>%
  read_cwl(format = "json") %>%
  is_cwl()
