Initialize teal_data object

Usage

new_teal_data(

  code = character(0),
  join_keys = join_keys(),

)

Arguments

data

(⁠named list⁠) of data objects.
code

(character or language) code to reproduce the data. Accepts and stores comments also.
join_keys

(join_keys) object
datanames

(character) names of datasets passed to data. Needed when non-dataset objects are needed in the env slot.
