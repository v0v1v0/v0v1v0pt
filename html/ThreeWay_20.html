nrm2 R Documentation

Columnwise normalization of a matrix

Description

Computation of a columnwise normalized version of a matrix.

Usage

 nrm2(A)

Arguments

A

Matrix of any order

Value

N

Matrix columnwise normalized

Examples

X <- matrix(rnorm(6*3),ncol=3)
Y <- nrm2(X)
apply(Y^2, 2, sum)
