chart_labs R Documentation

Chart labs

Description

Chart labs

Usage

chart_labs(.chart, title = NULL, x = NULL, y = NULL)

Arguments

.chart

A chart htmlwidget.
title

Text for main title.
x

Text for x-axis title.
y

Text for y-axis title.

Value

A chart htmlwidget.

Examples

chart(mtcars, caes(x = mpg, y = wt), type = "scatter") %>%
  chart_labs(
    title = "Main title",
    x = "X axis",
    y = "Y axis"
  )
