is_taxon_authority R Documentation

Check if is a taxon_authority

Description

Check if an object is of the taxon_authority class

Usage

is_taxon_authority(x)

Arguments

x

An object to test

Examples

x <- taxon_authority(c('Cham. & Schldl.', 'L.'),
                     date = c('1827', '1753'))
is_taxon_authority(x)
is_taxon_authority(1:3)
