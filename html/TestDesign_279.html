getLBUBInConstraintData R Documentation

(Internal) Parse item/stimulus lower/upper bounds from constraints data

Description

getLBUBInConstraintData is an internal function for parsing maximum item/stimulus lower/upper bounds from constraints data.

Usage

getLBUBInConstraintData(
  constants,
  constraints,
  item_constraints,
  stim_constraints
)

Arguments

constants

a names list containing constants.
constraints

a data.frame containing constraints data.
item_constraints

row numbers indicating which are item-level constraints.
stim_constraints

row numbers indicating which are stimulus-level constraints.

Value

getLBUBInConstraintData returns an updated list of constants.
