tv_images R Documentation

Retrieve TV series images.

Description

Get the images (posters and backdrops) for a TV series.

Usage

tv_images(api_key, id, language = NA, include_image_language = NA)

Arguments

api_key

Your TMDb Consumer Key.

id

The TV series ID.

language

A ISO 639-1 code.

include_image_language

A comma separated, a valid ISO 69-1. Maximum 5 per request.

Value

A list with the following fields:

id

The TV show ID.
backdrops

The backdrops of the TV show.
posters

Heigth, width, votes of the images of the TV show.

Author(s)

Andrea Capozio

References
