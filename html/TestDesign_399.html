parsePriorParameters R Documentation

(Internal) Parse prior parameters from viable sources

Description

parsePriorParameters is an internal function for parsing prior parameters from viable sources

Usage

parsePriorParameters(
  config_theta,
  simulation_constants,
  prior_density_override,
  prior_par_override
)

Arguments

config_theta

a list containing theta estimation configurations.
simulation_constants

a named list containing simulation constants.
prior_density_override

(optional) if supplied, treated as RAW distribution densities and overrides config values.
prior_par_override

(optional) if supplied, treated as NORMAL distribution parameters and overrides config values.

Value

parsePriorParameters returns a named list containing:

prior_dist

the type of distribution. One of NORMAL, UNIFORM, RAW.

prior_par

an examinee-wise list containing prior parameters.
