Description

bind ImageCollections

Usage

bind_ics(x)

Arguments

x

list of tidyee objects

Value

tidyee object containing single image collection and vrt

Examples

## Not run: 
library(tidyrgee)
library(rgee)

group_split(month)

  bind_ics()
