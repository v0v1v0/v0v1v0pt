rev R Documentation

Reverse a 'timeSeries'

Description

Reverses an uni- or multivariate "timeSeries" object.

Usage

## S3 method for class 'timeSeries'
rev(x)

Arguments

x

an uni- or multivariate "timeSeries" object.

Value

a "timeSeries" object

Examples


## Create Dummy "timeSeries"
tS <- dummyMonthlySeries()
   
## reverse series
rev(tS)
