get_telraam_token R Documentation

Get the current authentication token for the 'Telraam' API

Description

Get the current authentication token for the 'Telraam' API

Usage

get_telraam_token()

Value

Token currently used, set by set_telraam_token()

Examples

my_token <- "MyTelraamToken"
set_telraam_token(my_token)
get_telraam_token()
