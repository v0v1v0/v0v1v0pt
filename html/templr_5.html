from01 R Documentation

Helper function to scale from [0,1] to [min,max]

Description

Helper function to scale from [0,1] to [min,max]

Usage

from01(X, inp)

Arguments

X

values to scale
inp

list containing 'min' and 'max' values

Value

X scaled in [inp$min, inp$max]

Examples

from01(data.frame(x=matrix(runif(10))),list(x=list(min=10,max=20)))
