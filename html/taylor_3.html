album_palettes R Documentation

Color palettes based on Taylor Swift's albums

Description

Premade color palettes based on Taylor Swifts album covers. For details on how to extend and shorten these palettes, or create your own color palette, see color_palette().

Usage

album_palettes

album_compare

Format

A list of length 15. Each element contains a color palette for one of Taylor Swift's studio albums. The list elements are named according to the name of the album.

An object of class taylor_color_palette (inherits from vctrs_vctr, character) of length 15.

Source

Colors derived from album covers using 'Colormind'.

See Also

color_palette()

Examples

album_palettes

album_compare

album_palettes$evermore
