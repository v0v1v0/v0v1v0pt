tif2freq R Documentation

Periods Per Year for Time Index Frequencies

Description

Returns the frequency of a ti object constructed from the current date with the given tif.

Usage

tif2freq(tif)

Arguments

tif

a tifName or tif code

Value

a number

See Also

tif, tifName, frequency

Examples

tif2freq("wmonday")
tif2freq("monthly")
tif2freq(tif(today()))
