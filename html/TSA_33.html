explode.s R Documentation

A simulated explosive AR(1) series

Description

A simulated AR(1) series with the AR(1) coefficient being 3.

Usage

data(explode.s)

Format

The format is: Time-Series [1:8] from 1 to 8: 0.63 0.64 3.72 12.67 39.57 ...

Examples

data(explode.s)
