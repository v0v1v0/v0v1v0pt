review R Documentation

Retrieve basic informations about a review.

Description

Get the full details of a review by ID.

Usage

review(api_key, id)

Arguments

api_key

Your TMDb Consumer Key.

id

The review ID.

Value

A list with the following fields:

id

The review ID.
author

The review's author.
content

The review'text.
iso_639_1

The review's language.
media_id

The ID of the reviewed object.
media_title

The object's title.
media_type

The type of the reviewed object.
url

The url of the review.

Author(s)

Andrea Capozio

References

https://developers.themoviedb.org/3/getting-started
