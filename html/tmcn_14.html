revUTF8 R Documentation

Revert UTF-8 string to Chinese character.

Description

Revert UTF-8 string to Chinese character.

Usage

revUTF8(string, utype = "R")

Arguments

string

A character vector.
utype

UTF-8 string type, the default is R type, such as "<U+XXXX>".

Value

A character vector.
