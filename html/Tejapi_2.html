Tejapi.api_key R Documentation

Query or set Tejapi API key

Description

Query or set Tejapi API key

Usage

Tejapi.api_key(api_key)

Arguments

api_key

Optionally passed parameter to set Tejapi api_key.

Value

Returns invisibly the currently set api_key.

Examples

## Not run: 
Tejapi.api_key('foobar')

## End(Not run)
