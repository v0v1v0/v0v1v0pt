RTDalpha R Documentation

RTD alpha coefficient

Description

RTDalpha calculates RTD alpha coefficient

Usage

 RTDalpha(R0, R100)

Arguments

R0

resistance at 0C
R100

resistance at 100C

Value

alpha

RTD alpha coefficient
