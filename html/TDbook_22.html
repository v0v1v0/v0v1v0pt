data-tree_hmptree R Documentation

Tree to display with multiple graphs for multi-dimensional data

Description

This tree is to display with multiple graphs, and by reproducing Fig.2 of (Morgan, Segata, and Huttenhower 2013), demonstrates the feature of adding multiple layers to present different types of data by ggtreeExtra.
