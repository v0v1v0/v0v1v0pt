Create all 2^n n-tuples of ab[1] and ab[2]

Description

Create all 2^n n-tuples of ab[1] and ab[2]

Usage

CreateAllBinaries(n = 1, ab = c(0, 1))

Arguments

n

An integer > 0
ab

A vector of length 2

Details

This function is for internal usage only.

Value

a n x 2^n matrix

Examples

 CreateAllBinaries(3,c(0,1))
