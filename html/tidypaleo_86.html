theme_paleo R Documentation

A Paleo-friendly ggplot2 theme

Description

Essentially, this is ggplot2::theme_bw() with a few modifications

Usage

theme_paleo(...)

Arguments

...

Passed to ggplot2::theme_bw()

Value

A complete ggplot2::theme()

Examples

library(ggplot2)

ggplot(mpg, aes(cty, hwy)) +
  geom_point() +
  theme_paleo()
