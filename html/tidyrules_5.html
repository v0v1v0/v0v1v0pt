as_ruleset R Documentation

Get a ruleset from a rulelist

Description

Returns a ruleset object

Usage

as_ruleset(rulelist)

Arguments

rulelist

A rulelist

Value

A ruleset

See Also

rulelist

Examples

model_class_party = partykit::ctree(species ~ .,
                                    data = palmerpenguins::penguins
                                    )
as_ruleset(tidy(model_class_party))
