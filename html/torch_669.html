Rsqrt

Usage

torch_rsqrt(self)

Arguments

self

(Tensor) the input tensor.

rsqrt(input, out=NULL) -> Tensor

Returns a new tensor with the reciprocal of the square-root of each of the elements of input.

\mbox{out}_{i} = \frac{1}{\sqrt{\mbox{input}_{i}}}
