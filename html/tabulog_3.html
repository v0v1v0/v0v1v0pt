formatter R Documentation

Formatters

Description

Get or set the formatter for a parser

Usage

formatter(x)

formatter(x) <- value

Arguments

x

parser
value

formatter function to be set

Value

The formatter attribute (should be a function) for the passed object (usually a parser object)

Examples

p <- parser('[0-9]+]')

# Default formatter
formatter(p)

# Set formatter
formatter(p) <- as.integer

# Custom formatter
formatter(p)
