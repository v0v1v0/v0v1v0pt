theme_dr R Documentation

theme_dr

Description

Dimensional reduction scatter plot axis theme

Usage

theme_dr(
  xlength = 0.3,
  ylength = 0.3,
  arrow = grid::arrow(length = unit(0.15, "inches"), type = "closed")
)

Arguments

xlength

length of x axis
ylength

length of y axis
arrow

arrow specification, as created by 'grid::arrow()'

Value

a theme object with shorten axes
