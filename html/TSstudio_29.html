ts_info R Documentation

Get the Time Series Information

Description

Returning the time series object main characteristics

Usage

ts_info(ts.obj)

Arguments

ts.obj

A time series object of a class "ts", "mts", "xts", or "zoo"

Value

Text

Examples


# ts object
data("USgas")
ts_info(USgas)

# mts object
data("Coffee_Prices")
ts_info(Coffee_Prices)

# xts object
data("Michigan_CS")
ts_info(Michigan_CS)
