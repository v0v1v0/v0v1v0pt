my_taboola_data R Documentation

Sample of digital marketing data from Taboola downloaded by means of the Windsor.ai API.

Description

A dataset containing sample Taboola data fetched from windsor.ai API. See more at: https://windsor.ai/

Usage

my_taboola_data

Format

A data frame with 164 rows and 5 variables:

campaign

name of the campaign

clicks

number of clicks

spend

spend data

impressions

impressions data

date

date

Source

https://windsor.ai/
