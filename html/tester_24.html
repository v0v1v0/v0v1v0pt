is_decimal R Documentation

Is decimal

Description

Test if is a decimal number

Usage

  is_decimal(x)

Arguments

x

an R object

Details

decimal is any number in the intervals (-1,0) and (0,1)

See Also

is_integer

Examples

is_decimal(0.01) # TRUE
is_decimal(-0.01) # TRUE
is_decimal(0) # FALSE
is_decimal(1) # FALSE
is_decimal(runif(5))
is_decimal(rnorm(5))

M = matrix(seq(-2, 2, length.out=10), 5, 2)
is_decimal(M)
