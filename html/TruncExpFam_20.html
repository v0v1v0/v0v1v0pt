genrtruncClass R Documentation

Generates an rtrunc-dispatchable class

Description

Matches a list of arguments to an rtrunc method

Usage

genrtruncClass(n, family, parms)

Arguments

n

sample size
family

distribution family
parms

list of parameters passed to rtrunc (through the ... element)

Value

A character string.
