add_label R Documentation

Add label

Description

Add label to a card.

Usage

add_label(card, color, name = NULL, ...)

Arguments

card

Card id.
color

Label color.
name

Label name; choosing different non-existing name will create new label. Defaults to NULL.
...

Additional arguments passed to create_resource().
