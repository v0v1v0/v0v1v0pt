to01 R Documentation

Helper function to scale from [min,max] to [0,1]

Description

Helper function to scale from [min,max] to [0,1]

Usage

to01(X, inp)

Arguments

X

values to scale
inp

list containing 'min' and 'max' values

Value

X scaled in [0,1]

Examples

to01(10+10*data.frame(x=matrix(runif(10))),list(x=list(min=10,max=20)))
