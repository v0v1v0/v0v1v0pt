toy_data R Documentation

Toy dataset

Description

A data frame with 9999 genetic associations

Usage

data(toy_data)

Format

A data frame with 9999 genetic associations

Details

  • rsID. SNP name

  • freq. allele frequency of effect SNP

  • A1_beta. effect size

  • Analysis. adds colour to the type of analysis (e.g. GWAS, Sequencing)

  • Gene. Candidate gene name

  • N.

  • trait. ToyDataPheno
