Dados7 R Documentation

Dados de exemplo de DQL (4 tratamentos comuns e 2 testemunhas).

Description

Exemplo simulado de um experimento em DQL com quatro tratamentos comuns qualitativos e duas testemunhas.

Usage

data(Dados7)

Format

Um data frame com 36 observacoes das seguintes variaveis.

Test

Vetor do tipo Fator.

Medicamento

Vetor com os tratamentos.

Animal

Vetor com a identificao das linhas.

Epoca

Vetor com a identificacao das colunas.

Carrapato

Vetor do tipo numerico com a variavel resposta.
