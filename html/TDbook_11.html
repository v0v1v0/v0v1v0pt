data-df_inode_data R Documentation

Nodedata to be mapped to tree

Description

This data contains annotation information about nodes of "tree_boots.nwk", that is to be mapped to the tree using the ⁠%<+%⁠ operator.
Chap.7.1

Format

A data frame with 6 rows and 6 variables with a column of node or first column of taxa labels.

Source

https://raw.githubusercontent.com/TreeViz/metastyle/master/design/viz_targets_exercise/inode_data.csv

Examples

data(df_inode_data)
