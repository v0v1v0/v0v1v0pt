list_get R Documentation

Get a list by id.

Description

Retrieve all the basic information about a list searched by ID.

Usage

list_get(api_key, id)

Arguments

api_key

Your TMDb Consumer Key.

id

The list ID.

Value

A list with the following fields:

created_by

The author of the list.
description

A brief description about the list.
favorite_count

The number of favorites.
id

The list ID.
items

The elements of the list.
item_count

The number of list's elements.
iso_639_1

The language of the list.
name

The name of the list.
poster_path

The poster of the list.

Author(s)

Andrea Capozio

References

https://developers.themoviedb.org/3/getting-started
