tempcont_data R Documentation

Dummy data for temp.cont examples

Description

Data from field work in the Middle-earth.

Usage

data("tempcont_data")

Format

A data frame with 400 observations on the following 10 variables.

cave

a factor vector

year

a numeric vector with a temporal variable

trolls

a numeric vector

bushes

a numeric vector

temperature

a numeric vector

rainfall

a numeric vector

drought

a numeric vector

gremlins

a numeric vector

orcs

a numeric vector

warlocks

a numeric vector

Examples

data(tempcont_data)
str(tempcont_data)
