ts_aug_none R Documentation

no augmentation

Description

Does not make data augmentation.

Usage

ts_aug_none()

Value

a ts_aug_none object.

Examples

library(daltoolbox)
data(sin_data)

#convert to sliding windows
xw <- ts_data(sin_data$y, 10)

#no data augmentation
augment <- ts_aug_none()
augment <- fit(augment, xw)
xa <- transform(augment, xw)
ts_head(xa)
