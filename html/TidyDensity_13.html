Cumulative Geometric Mean

Description

A function to return the cumulative geometric mean of a vector.

Usage

cgmean(.x)

Arguments

.x

A numeric vector

Details

A function to return the cumulative geometric mean of a vector. exp(cummean(log(.x)))

Value

A numeric vector
