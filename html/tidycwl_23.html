Is this CWL v1.1?

Description

Is this CWL v1.1?

Usage

is_v1.1(x)

Arguments

x

CWL object

Value

Logical. TRUE if it is a CWL v1.1 object, FALSE if not.

Examples

system.file("cwl/sbg/workflow/rnaseq-salmon.json", package = "tidycwl") %>%
  read_cwl(format = "json") %>%
  is_v1.1()
