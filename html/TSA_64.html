prey.eq R Documentation

Prey series / time series

Description

The stationary part of the Didinium series in the veilleux data frame.

Usage

data(prey.eq)

Format

The format is: Time-Series [1:57] from 7 to 35: 26.9 53.2 65.6 81.2 143.9 ...
