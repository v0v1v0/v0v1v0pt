nhefs_weights R Documentation

NHEFS with various propensity score weights

Description

A dataset containing various propensity score weights for causaldata::nhefs_complete.

Usage

nhefs_weights

Format

A data frame with 1566 rows and 14 variables:

qsmk

Quit smoking

race

Race

age

Age

education

Education level

smokeintensity

Smoking intensity

smokeyrs

Number of smoke-years

exercise

Exercise level

active

Daily activity level

wt71

Participant weight in 1971 (baseline)

w_ate

ATE weight

w_att

ATT weight

w_atc

ATC weight

w_atm

ATM weight

w_ato

ATO weight
