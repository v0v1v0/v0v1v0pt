reinforcement_tm R Documentation

Reinforcement_tm

Description

The reinforcement_tm-function computes Robin and Alexander's (2004) 4-cycle metric for two-mode networks.

Usage

reinforcement_tm(net)

Arguments

net

A two-mode network

Value

Returns the score

Note

version 1.0.0

Author(s)

Tore Opsahl; http://toreopsahl.com

References

http://toreopsahl.com/tnet/two-mode-networks/clustering/

Examples

## Load sample data
net <- rg_tm(10, 7, 0.4)

## Run the programme
reinforcement_tm(net)
