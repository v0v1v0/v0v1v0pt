has_endmark R Documentation

Test for Incomplete Sentences

Description

A logical test of missing sentence ending punctuation.

Usage

has_endmark(x, endmarks = c("?", ".", "!"), ...)

Arguments

x

A character vector.
endmarks

The potential ending punctuation marks,
...

ignored.

Value

Returns a logical vector.

Examples

x <- c(
    "I like it.", 
    "Et tu?",  
    "Not so much", 
    "Oh, I understand.",  
    "At 3 p.m., we go",
    NA
)
has_endmark(x)
