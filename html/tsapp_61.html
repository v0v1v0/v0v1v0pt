perwinda Daniell window for direct spectral estimation

Description

perwinda Daniell window for direct spectral estimation

Usage

perwinda(e, n)

Arguments

e

equal bandwidth (at most n frequencies are used for averaging)
n

length of time series

Value

w weights (symmetric)

Examples

data(WHORMONE)
w <- perwinda(0.1,length(WHORMONE))
