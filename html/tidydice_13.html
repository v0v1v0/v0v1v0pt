plot_single_coin R Documentation

Draw a single coin

Description

Draw a single coin

Usage

plot_single_coin(
  ggplot = NULL,
  result = 1,
  x = 0,
  y = 0,
  width = 0.9,
  fill = "white",
  detailed = FALSE,
  line_size = 0.8,
  line_color = "black"
)

Arguments

ggplot

ggplot-Object. If passed, the dice will be added to plot
result

Result of flip coin (0/1)
x

X-coordinate of dice (center)
y

y-coordinate of dice (center)
width

Width of coin
fill

Fill color
detailed

If TRUE, the dice is plotted with more details
line_size

Size of Lines
line_color

Color of Lines

Value

ggplot-Object
