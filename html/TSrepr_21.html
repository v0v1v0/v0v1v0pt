MSE

Description

The mse computes MSE (Mean Squared Error) of a forecast.

Usage

mse(x, y)

Arguments

x

the numeric vector of real values
y

the numeric vector of forecasted values

Value

the numeric value

Author(s)

Peter Laurinec, <tsreprpackage@gmail.com>

Examples

mse(runif(50), runif(50))
