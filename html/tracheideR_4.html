tch R Documentation

Raw light intensity for two years (2010 and 2013) of Pinus pinaster

Description

This dataset has 2 years (2010 and 2013) each one with 3 raw profiles (in grayscale).

Usage

data("tch")

Format

A list with 2 elements:

a y2010

a list with 3 radii

a y2013

a list with 3 radii
