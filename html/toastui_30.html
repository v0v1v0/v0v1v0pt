cal_template R Documentation

Set template for a calendar

Description

Template JS functions to support customer renderer

Usage

cal_template(
  cal,
  milestoneTitle = NULL,
  taskTitle = NULL,
  alldayTitle = NULL,
  ...
)

Arguments

cal

A calendar() object.
milestoneTitle

The milestone title (at left column) template function.
taskTitle

The task title (at left column) template function.
alldayTitle

The allday title (at left column) template function.
...

Additionals arguments, see online documentation.

Value

A calendar htmlwidget object.

Note

Online JavaScript documentation: https://github.com/nhn/tui.calendar/blob/main/docs/en/apis/template.md. All arguments must be JavaScript function with htmlwidgets::JS().

Examples

calendar(view = "week", taskView = TRUE) %>%
  cal_template(
    milestoneTitle = "TODO",
    taskTitle = "Assignment",
    alldayTitle = "Full-time"
  )
