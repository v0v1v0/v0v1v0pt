sendVideo R Documentation

sendVideo

Description

Send mp4 videos

Usage

sendVideo(video = NULL, duration = NULL, caption = NULL,
  reply_to_message_id = NULL, chat_id = NULL)

Arguments

video

Video to send (required)
duration

Duration of sent video in seconds
caption

Video caption
reply_to_message_id

If the message is a reply, ID of the original message
chat_id

Unique identifier for the target chat or username of the target channel (required)
