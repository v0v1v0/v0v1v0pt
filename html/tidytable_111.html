pick R Documentation

Selection version of across()

Description

Select a subset of columns from within functions like mutate(), summarize(), or filter().

Usage

pick(...)

Arguments

...

Columns to select. Tidyselect compatible.

Examples

df <- tidytable(
  x = 1:3,
  y = 4:6,
  z = c("a", "a", "b")
)

df %>%
  mutate(row_sum = rowSums(pick(x, y)))
