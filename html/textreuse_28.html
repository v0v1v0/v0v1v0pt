TextReuseTextDocument-accessors R Documentation

Accessors for TextReuse objects

Description

Accessor functions to read and write components of TextReuseTextDocument and TextReuseCorpus objects.

Usage

tokens(x)

tokens(x) <- value

hashes(x)

hashes(x) <- value

minhashes(x)

minhashes(x) <- value

Arguments

x

The object to access.
value

The value to assign.

Value

Either a vector or a named list of vectors.
