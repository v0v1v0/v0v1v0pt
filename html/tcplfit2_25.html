gnlsderivobj R Documentation

GNLS Derivative Objective Function

Description

Derivative of the gnls function set to zero for top location solver.

Usage

gnlsderivobj(x, tp, ga, p, la, q)

Arguments

x

Concentration.
tp

Top.
ga

Gain AC50.
p

Gain power.
la

Loss AC50.
q

Loss power.

Value

Value of gnls derivative at x.
