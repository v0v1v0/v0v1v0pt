Usage

(Tensor) the input tensor.

expm1(input, out=NULL) -> Tensor

Returns a new tensor with the exponential of the elements minus 1 of input.

y_{i} = e^{x_{i}} - 1

Examples

if (torch_is_installed()) {

torch_expm1(torch_tensor(c(0, log(2))))
