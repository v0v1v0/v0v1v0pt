Description

Computation of the matrix with rows being the linearized matrix products of X[,,i] and B.

Usage

X %w% B

Arguments

X

A numerical 3D array. Each slice is a matrix of column size the same as the row size of B.
B

A numerical matrix.

Value

numerical 3D array.
