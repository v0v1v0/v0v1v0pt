clear_stack R Documentation

Remove all objects from the stack by deleting them from memory.

Description

Remove all objects from the stack by deleting them from memory.

Usage

clear_stack()

Examples

library(dplyr)
aframe <- data.frame(zed = runif(100))
set_to_zero <- . %>% mutate(zed = 0)
aframe %>% scion(zed >0.5, false_fun=set_to_zero) 
clear_stack()
