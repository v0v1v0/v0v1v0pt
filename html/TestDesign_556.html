validateExpression R Documentation

(Internal) Validate constraint condition expression

Description

validateExpression is an internal function for validating a constraint's condition expression.

Usage

validateExpression(x, attrib, unit_name, use_lt)

Arguments

x

a data.frame row containing a single constraint data.
attrib

an item_attrib object or a st_attrib object.
unit_name

items or stimuli.
use_lt

if TRUE, will raise an error when number of matching items/stimuli is less than 2. if FALSE, will raise an error when number of matching items/stimuli is 0.

Value

validateExpression does not return any values; it stops if the input constraint is not valid.
