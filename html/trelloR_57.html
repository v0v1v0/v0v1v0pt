Get Member Info

Description

Returns a flat data frame with member-related data.

Usage

get_member_fields(
  id,
  fields = c("fullName", "username", "memberType", "bio"),
  ...
)

Arguments

id

member ID or username
fields

by default fetches fullName, username, memberType, bio
...
