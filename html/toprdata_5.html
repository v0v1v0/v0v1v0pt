toprdata R Documentation

toprdata

Description

Gene and exon datasets to use with the 'topr' package

datasets

The two datasets are:

  • ENSGENES Gene information from genome build GRCh38

  • ENSEXONS Exon information from genome build GRCh38

  • ENSGENES_37 Gene information from genome build GRCh37

  • ENSEXONS_37 Exon information from genome build GRCh37

Examples

library(toprdata)
head(ENSGENES)
head(ENSEXONS)
