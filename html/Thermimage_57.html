rotate270.matrix R Documentation

Rotate a matrix by 270 degrees counterclockwise (or 90 degree clockwise). Used for adjusting image plotting in R.

Description

Rotate a matrix by 270 degrees counterclockwise (or 90 degree clockwise). Used for adjusting image plotting in R.

Usage

rotate270.matrix(x)

Arguments

x

A matrix corresponding to raster or image data.

Value

Returns a matrix

Author(s)

Glenn J Tattersall
