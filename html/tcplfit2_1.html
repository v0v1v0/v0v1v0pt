acgnlsobj R Documentation

AC GNLS Objective Function

Description

GNLS objective function set to y for gnls solver.

Usage

acgnlsobj(x, y, tp, ga, p, la, q)

Arguments

x

Concentration.
y

Desired activity level.
tp

Top.
ga

Gain AC50.
p

Gain power.
la

Loss AC50.
q

Loss power.

Value

Difference between GNLS model repsone at x and y.
