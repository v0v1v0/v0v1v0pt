drop_na R Documentation

Drop rows containing missing values

Description

Drop rows containing missing values

Usage

drop_na(.df, ...)

Arguments

.df

A data.frame or data.table
...

Optional: A selection of columns. If empty, all variables are selected. tidyselect compatible.

Examples

df <- data.table(
  x = c(1, 2, NA),
  y = c("a", NA, "b")
)

df %>%
  drop_na()

df %>%
  drop_na(x)

df %>%
  drop_na(where(is.numeric))
