tm1_logout R Documentation

TM1 Log Out

Description

Logs out

Usage

tm1_logout(tm1_connection)

Arguments

tm1_connection

tm1 connection object returned by the function tm1_connection

Examples

## Not run: 
tm1_logout(tm1_connection("localhost", "8881", "admin", "apple"))

con_obj <- tm1_connection("localhost", "8881", "admin", "apple")
tm1_logout(con_obj)

## End(Not run)
