label R Documentation

Label attribute.

Description

Label attribute.

Usage

label(x)

label(x) <- value

setLabel(x, value)

has.label(x)

Arguments

x

An object.
value

A character specifying the label.

Functions

  • label(x) <- value: Set label attribute.

  • setLabel(): Set label attribute.

  • has.label(): Check for label attribute.

Examples

x <- 1:10
label(x) <- "Foo"
x <- setLabel(x, "Foo") # Alternative syntax
has.label(x)
label(x)
