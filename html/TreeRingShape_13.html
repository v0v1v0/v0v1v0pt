Lrn R Documentation

Return a ring number of tree ring polygons list (L) from year

Description

Return a ring number of tree ring polygons list (L) from year

Usage

Lrn(L, yr)

Arguments

L

tree ring polygons list (L)
yr

years (or rings)

Value

a ring number of tree ring polygons list (L)

Examples

Lrn(TR@L,168)  # 168 is the formation year (from outermost) of the tree ring
