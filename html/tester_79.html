is_single_number R Documentation

Is single number

Description

Tests if an object is a single number

Usage

  is_single_number(x)

Arguments

x

an R object

See Also

is_single

Examples

is_single_number(1.0)  # TRUE
is_single_number("hoskdflksfd")  # FALSE
is_single_number("1.0")  # FALSE
is_single_number(1:5)  # FALSE
