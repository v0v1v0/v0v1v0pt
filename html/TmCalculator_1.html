c2s R Documentation

convert a vector of characters into a string

Description

Simply convert a vector of characters such as c("H","e","l","l","o","W","o","r","l","d") into a single string "HelloWorld".

Usage

c2s(characters)

Arguments

characters

A vector of characters

Value

Retrun a strings

Examples


c2s(c("H","e","l","l","o","W","o","r","l","d"))
