twn_statuscodes R Documentation

TWN statuscodes

Description

De TWN-lijst heeft aan ieder taxon een statuscode toegekend. De betekenis van deze codes is opgenomen in de tabel twn_statuscodes

Usage

twn_statuscodes

Format

Dataframe met 2 kolommen

  • status - TWN-statuscode

  • omschrijving- Omschrijving van de betekenis van de statuscode

Source

https://www.aquo.nl/index.php/Vraag_en_Antwoord_TWN

Examples


twn_statuscodes
