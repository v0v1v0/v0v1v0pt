ima22.s R Documentation

Simulated IMA(2,2) series / time series

Description

A simulated IMA(2,2) series with theta1=1 and theta2=-0.6

Usage

data(ima22.s)

Format

The format is: Time-Series [1:62] from 1 to 62: 0.00000 0.00000 -0.00569 2.12404 2.15337 ...

Examples

data(ima22.s)
