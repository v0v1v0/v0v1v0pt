noctula R Documentation

Nyctalus noctula echolocation data

Description

Echolocation signal for the Nyctalus noctula hunting bat

Usage

data("noctula")

Format

The format is: num [1:96] -18 16 -5 -17 21 -6 -17 20 -6 -16 ...
