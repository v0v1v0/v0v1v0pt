helpGC R Documentation

Extensive Help Via Vignettes

Description

A convenience function to show vignettes associated with package tigerstats. Vignette will open in the user's default browser.

Usage

helpGC(topic)

Arguments

topic

filename of the vignette, exclusive of the .html extension

Value

side effects

Author(s)
