isGBK R Documentation

Indicate whether the encoding of input string is GBK.

Description

Indicate whether the encoding of input string is GBK.

Usage

isGBK(string, combine = FALSE)

Arguments

string

A character vector.
combine

Whether to combine all the strings.

Value

Logical value.

Author(s)

Jian Li <rweibo@sina.com>

Examples

isGBK("hello")
