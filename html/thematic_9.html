okabe_ito R Documentation

A color-blind safe qualitative colorscale (Okabe-Ito)

Description

This is the default qualitative colorscale in thematic_on()

Usage

okabe_ito(n = NULL)

Arguments

n

number of colors.

Value

a vector of color codes.
