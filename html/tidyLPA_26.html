poms R Documentation

Apply POMS-coding to data

Description

Takes in a data.frame, and applies POMS (proportion of of maximum)-coding to the numeric columns.

Usage

poms(data)

Arguments

data

A data.frame.

Value

A data.frame.

Author(s)

Caspar J. van Lissa

Examples

data <- data.frame(a = c(1, 2, 2, 4, 1, 6),
                   b = c(6, 6, 3, 5, 3, 4),
                   c = c("a", "b", "b", "t", "f", "g"))
poms(data)
