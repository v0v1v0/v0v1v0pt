is.decorated R Documentation

Decorated Functions

Description

Returns TRUE if the function f is decorated, otherwise FALSE.

Usage

is.decorated(f)

Arguments

f

A function.

Value

TRUE or FALSE.

Examples

source_decoratees(tinsel_example('timer.R'))

# sourced from the timer.R example file
is.decorated(waldo)
is.decorated(jack)

# it's a function, but not decorated
is.decorated(mean)

# far from the mark
is.decorated(3030)
