eco.cd6 R Documentation

6-month rates

Description

6-month rates 1/1/1991 through 4/1/2010

Usage

data("eco.cd6")

Format

The format is: num [1:469] 7.25 7.53 7.64 7.64 7.59 7.44 7.39 7.26 7.25 7.19 ...

Source

Internet

References
