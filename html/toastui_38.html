chart_options R Documentation

Chart options

Description

Chart options

Usage

chart_options(.chart, ...)

Arguments

.chart

A chart htmlwidget.
...

Named list of options, options depends on chart's type, see common options here.

Value

A chart htmlwidget.

Examples

chart(mtcars, caes(x = mpg, y = wt), type = "scatter") %>% 
  chart_options(
    chart = list(title = "A scatter chart")
  )
