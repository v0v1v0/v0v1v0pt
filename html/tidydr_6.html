element_line2 R Documentation

element_line2

Description

element_line2 for drawing shorten axis lines

Usage

element_line2(
  colour = NULL,
  size = NULL,
  linetype = NULL,
  lineend = NULL,
  color = NULL,
  arrow = NULL,
  inherit.blank = FALSE,
  id,
  xlength = 0.3,
  ylength = 0.3,
  ...
)

Arguments

colour

line colour
size

line size in pts
linetype

line type
lineend

line end style (round, butt, square)
color

aliase to colour
arrow

arrow specification, as created by 'grid::arrow()'
inherit.blank

whether inherit 'element_blank'
id

1 or 2, 1 for axis.line.x.bottom and 2 for axis.line.y.left, only these two axes supported
xlength

length of x axis
ylength

length of y axis
...

additional parameters

Value

element_line2 object, which is a tailored element_line object
