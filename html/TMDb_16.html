genres_movie_list R Documentation

Retrieve the available genres for movies.

Description

Get the list of movie genres.

Usage

genres_movie_list(api_key, language = NA)

Arguments

api_key

Your TMDb Consumer Key.

language

A ISO 639-1 code.

Value

A single list item, with the following fields:

id

The ID of a movie genre.
name

The name of a movie genre.

Author(s)

Andrea Capozio

References

https://developers.themoviedb.org/3/getting-started

Examples
