my_tiktokads_data R Documentation

Sample data of TikTok ads from the Windsor API.

Description

A dataset containing sample TikTok ads data fetched from windsor.ai API. See more at: https://windsor.ai/

Usage

my_tiktokads_data

Format

A data frame with 164 rows and 5 variables:

campaign

name of the campaign

clicks

number of clicks

spend

spend data

impressions

impressions data

date

date

Source

https://windsor.ai/
