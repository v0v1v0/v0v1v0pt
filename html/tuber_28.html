get_channel_stats R Documentation

Get statistics of a Channel

Description

Get statistics of a Channel

Usage

get_channel_stats(channel_id = NULL, mine = NULL, ...)

list_my_channel(...)

Arguments

channel_id

Character. Id of the channel
mine

Boolean. TRUE if you want to fetch stats of your own channel. Default is NULL.
...

Additional arguments passed to tuber_GET.

Value

nested named list with top element names: kind, etag, id, snippet (list of details of the channel including title), statistics (list of 5)

If the channel_id is mistyped or there is no information, an empty list is returned

References

https://developers.google.com/youtube/v3/docs/channels/list
