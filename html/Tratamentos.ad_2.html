Dados1 R Documentation

Dados de exemplo (1 fator e 1 testemunha).

Description

Exemplo simulado de um experimento com um fator qualitativo, uma testemunha adicional e 4 repeticoes. "Simulated example of an experiment with an qualitative factor, additional control and 4 repetitions".

Usage

data(Dados1)

Format

Um data frame com 20 observacoes das seguintes variaveis.

Test

Vetor do tipo Fator.

Espacamento

Vetor do tipo Fator.

Rep

Vetor numerico.

Prod

Vetor numerico.
