taper R Documentation

taper taper modification of a time series

Description

taper taper modification of a time series

Usage

taper(y, part)

Arguments

y

the time series
part

scalar, 0 <= part <= 0.5, part of modification (at each end of y)

Value

tp tapered time series

Examples

data(WHORMONE)
out <-taper(WHORMONE,0.3)
 
plot(WHORMONE) 
lines(out,col="red")
