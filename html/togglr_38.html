toggl_start R Documentation

toggl_start

Description

start a task

Usage

toggl_start(
  description = get_context(),
  client = "without client",
  project_name = get_context_project(),
  start = now(),
  api_token = get_toggl_api_token(),
  tags = "",
  workspace_id = get_workspace_id(api_token)
)

Arguments

description

the task you are doing
client

client name
project_name

nom du projet
start

start time in POSIXt
api_token

the toggl api token
tags

tags
workspace_id

workspace id

Examples

## Not run: 
get_toggl_api_token()# set your api token here
toggl_start()

## End(Not run)
