myGS R Documentation

Orthogonal projection of y onto x

Description

Orthogonal projection of y onto x.

Usage

myGS(x,y)

Arguments

x,y

a pairs of vectors.

Value

myGS returns a vector of the projection of y onto x.

See Also

G3S

Examples

x <-  sample(c((-10):10),3)
y <-  sample(c((-10):10),3)
myGS(x,y)
fractional(myGS(x,y))
