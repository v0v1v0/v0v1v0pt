pcm_to_wav R Documentation

Convert PCM to WAV

Description

Accepts PCM audio data as input and generates a corresponding WAV file

Usage

pcm_to_wav(
  input,
  output = tempfile(fileext = ".wav"),
  sample_rate = 16000,
  extensible = FALSE
)
