person_tv_credits R Documentation

Retrieve person TV credits.

Description

Get the TV credits for a specific person id.

Usage

person_tv_credits(api_key, id, language = NA, append_to_response = NA)

Arguments

api_key

Your TMDb Consumer Key.

id

The person ID.

language

A ISO 639-1 code.

append_to_response

Comma separated, any person method.

Value

A list with the following fields:

cast

The tv shows in which the person has played.
crew

The tv shows in which the person has had crew roles.
id

The person ID.

Author(s)

Andrea Capozio

References

https://developers.themoviedb.org/3/getting-started
