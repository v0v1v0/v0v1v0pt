tm1_create_subset R Documentation

TM1 Create New Subset to a Dimension

Description

Creates a new subset to the dimension

Usage

tm1_create_subset(tm1_connection,
  dimension, subset, element="", mdx="", overwrite=TRUE)

Arguments

tm1_connection

tm1 connection object returned by the function tm1_connection
dimension

Name of dimension
subset

Name of new subset
element

Name of elements seperated by | for static subset
mdx

mdx of subset for dynamic subset
overwrite

TRUE or FALSE. If TRUE, subset is overwritten

Examples

## Not run: 
tm1_create_subset(tm1_connection("localhost", "8881", "admin", "apple"),
"month", "Q1Months", element = "Jan|Feb|Mar")

con_obj <- tm1_connection("localhost", "8881", "admin", "apple")
tm1_create_element(con_obj, "month", "all", mdx = "[month].MEMBERS")

## End(Not run)
